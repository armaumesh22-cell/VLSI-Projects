# 4-bit Up/Down Counter using Verilog

This project implements a 4-bit up/down counter using Verilog.

## Features
- Counts up when control signal is high
- Counts down when control signal is low
- Reset functionality

## Description
The counter updates its value on every clock edge.
It increments or decrements based on the control input signal.

## Tools Used
- Verilog
- ModelSim / GTKWave

## Learning Outcome
This project helped me understand sequential logic and counter design.
