# Traffic Light Controller using Verilog

This project implements a traffic light controller using a Finite State Machine (FSM).

## States
- RED
- GREEN
- YELLOW

## Description
The system transitions between different traffic light states based on a clock signal and timing conditions.
Each state is maintained for a fixed number of clock cycles.

## Features
- FSM-based design
- Sequential logic implementation
- Clock and reset control

## Tools Used
- Verilog
- ModelSim / GTKWave

## Learning Outcome
This project helped me understand FSM design and sequential circuits in Verilog.
